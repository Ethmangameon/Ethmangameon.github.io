Privacy Policy

Thank you for using Alice (the “Bot”, “AI”, “Service”, “App”, “Website”) created by Stuff and Things (“Us”, “We”, “Our”, “Ours”,). To write message context histories, help development of Alice, and check if a message is directed towards Alice, we collect data from the Discord User(s) (“You”, “They”, “Their”,) who are in the same Discord Guild(s) that Alice has been added in.

Your privacy is important to us, and we have designed this Privacy Policy to outline what data we collect and store via our software.

 

Data We Collect

We collect and store the following data in order to deliver the best possible experience to our Users.

  • User ID: The unique identifier which belongs to each user on Discord. This allows Alice to @mention a specific User and customize initial prompt per user.
  • Username: The Username of a User on Discord, used to add personalization to Alice in its messages. 
  • User Tag: The Nickname of a User on a Guild, used to add personalization to Alice in its messages, unique to a certain guild.
  • Guild Name: The name of a Discord Guild, used to add personalization to Alice in the messages it sends, which is unique to your Guild.
  • Channel ID: The unique identifier which belongs to each Discord Channel. This allows Alice to mention the channel and to have custom personalization of the bot per channel.
  • Channel Names: The name given to a channel of a Discord Guild. This gives a more personalized experience with Alice.
  • Message History: The history of messages sent into a Guild with Alice added as an interaction. Used to help validate Terms of Service.
  • Messages: Messages sent into a Guild. Messages are checked to see if they are on an authorized channel, and to see if it is directed towards Alice. In the event a message is sent towards Alice, Alice will use the message as a prompt to an LLM, and will cache that message and response in a tokenized file associated with the channel ID.
  • Message ID’s: The unique identifier given to a message sent on Discord. Used for distinguishing between messages sent through a Guild that Alice has been added to.
  • Text File Attachments on Messages: Text files sent through Discord. If it is directed towards Alice, it is used the same way as messages, as a prompt to an LLM, and then cached in a tokenized file associated with the channel ID.
  • Images Attachments on Interactions: Images sent with messages on Discord. If a command is used for Alice which involves an image, it stores an image temporarily into memory and encodes it into base64, then it will send the encoded image to an API to be processed how the user requested.
  • Initial Prompts: A message which is added to the beginning of every prompt a User sends to Alice. Each is unique to a certain user and is stored for use in every message Alice responds to a certain user ID which the initial prompt is tied to.
  • System Messages: Specified personality and guidelines for Alice to follow. Aids in generating messages from Alice that are relevant to what system message is set to. Tied to Channel ID.

 

We (Stuff and Things) do not sell any data we collect from Alice and any Guilds that Alice is in to advertisers or any other third parties.

If there are any questions or about privacy, please contact us at our Discord server linked above.
